## Button 新增属性

| 属性名       | 说明         | 类型   | 默认值 |
| ------------ | ------------ | ------ | ------ |
| throttleWait | 节流等待时间 | number | 0      |
| debounceWait | 防抖等待时间 | number | 0      |
